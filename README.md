# 富兰克林十三美德 BINGO

一个可直接在浏览器打开的中文 Bingo 小游戏。主题基于 Benjamin Franklin 的十三美德，支持简单任务、困难任务、任务管理、自定义增删改、进度圈、BINGO 烟花和对角线连线判定。

## 本地打开

直接打开 `index.html` 即可。

## 文件结构

- `index.html`：页面结构
- `styles.css`：视觉样式
- `script.js`：游戏逻辑

## 分享到 GitHub Pages

把这几个文件推到 GitHub 仓库后，在仓库的 `Settings > Pages` 中启用 Pages，选择 `Deploy from a branch`，分支选 `main`，目录选 `/ (root)`。
